Grid

Модульная сетка

Использование

В компоненте предусмотрены различные способы передачи дочерних элементов, но если нет и не удаётся найти подходящего решения для какого-то кейса, то можно обойтись без pug-шаблона и использовать только стилевой файл компонента

Базовый компонент

Показать шаблон 
mixin grid (props = {})
-
  const
        tag = props.tag || "div",
        content = props.content || [],
        modifiers = props.modifiers ? props.modifiers.map((mod) => "grid--" + mod) : [];

case tag
  when "ul"
  when "ol"
    #{tag}(class=["grid", ...modifiers])&attributes(attributes)
      each item in content
        - const itemModifiers = item.modifiers ? item.modifiers.map((mod) => "grid__item--" + mod) : [];

        li(class=["grid__item", ...itemModifiers])
          +#{item.component}(item.params)

  default
    #{tag}(class=["grid", ...modifiers])&attributes(attributes)
      if content.length
        each item in content
          - const itemModifiers = item.modifiers ? item.modifiers.map((mod) => "grid__item--" + mod) : [];
          - const itemParentClass = item.parentClass || [];

          +#{item.component}(item.params)(class=["grid__item", ...itemModifiers, ...itemParentClass])
      else
        block
Показать стили 
.grid {
  display: grid;
  grid-template-columns: repeat(var(--cols, 1), minmax(0, 1fr));

  &ul,
  &ol {
    margin: 0;
    padding: 0;
    list-style: none;
  }

  &__item {
    grid-column: var(--col-start, span) var(--col-end, 1);
    grid-row: var(--row-start, span) var(--row-end, 1);
  }
}

Модификаторы

Если для проекта не предусмотрены токены, используйте жёстко закодированные значения

Стили для адаптива задавайте как удобней:
- переопределением в самом модификаторе, например: 

&--cols {
  &-2 {
    --cols: 2; 

    @include vp-767 {
      --cols: 1;
    }
}

- переопределением от родительского класса .parent__grid или .parent .grid
- отдельными модификаторами с отражением в имени как меняется раскладка, а не в каком блоке/компоненте гриды используются

.grid {
  ...
