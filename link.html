Link

Ссылка

Использование

Заблокированные и находящиеся в активном состоянии интерактивные элементы не должны попадать в фокус при навигации с клавиатуры. Для этого ссылкам с тегом <a> добавляется атрибут tab-index="-1", с тегом <button> -- атрибут disable. Если не нужно поддерживать Safari 15, вместо tab-index="-1" можно использовать атрибут inert

Компоненту задаются стили по умолчанию и затем по необходимости переопределяются

Не стоит использовать свойство opacity без крайней необходимости, т.к. это может исказить визуал (при использовании кнопки поверх цветных элементов) и ухудшить читаемость при смене цветовой темы. Вместо opacity по возможности лучше использовать альфа-канал

Примеры

Заблокированная ссылка

Добавляет класс .is-disabled и атрибут tabindex="-1" или disabled

Показать код 
+link({
    isDisabled: true,
    text: "Link"
  })

Ссылка в активном состоянии

Добавляет класс .is-active и атрибут tabindex="-1" или disabled

Показать код 
+link({
    isActive: true,
    text: "Link"
  })

Ссылка с иконкой справа

Показать код 
+link({
    text: "Link",
    icon: {
      name: 'icon-arrow',
      width: 10,
      height: 16,
    },
  })

Ссылка с иконкой слева

Показать код 
+link({
    text: "Link",
    icon: {
      name: 'icon-arrow',
      width: 10,
      height: 16,
      isLeft: true,
    },
  })

Базовый компонент

Показать шаблон 
mixin link(props = {})
  -
    const
          tag = props.tag ? props.tag : props.href ? "a" : "button",
          href = props.href,
          text = props.text || "",
          type = props.type || tag === "button" && "button",
          target = props.isExternal && "_blank",
          rel = props.isExternal && "nofollow noopener noreferrer",
          icon = props.icon,
          {name, width, height, isLeft} = icon || {},
          isActive = props.isActive,
          activeClass = isActive && "is-active",
          isDisabled = props.isDisabled,
          disabledClass = isDisabled && "is-disabled",
          disabledAttr = (isDisabled || isActive) && tag === "button" && "true",
          tabindexAttr = props.tabindex || ((isDisabled || isActive) && tag === "a" && "-1"),
          modifiers = props.modifiers ? props.modifiers.map((mod) => "link--" + mod) : [];

  #{tag}(class=["link", disabledClass, activeClass, ...modifiers], href=href, target=target, rel=rel, type=type, disabled=disabledAttr, tabindex=tabindexAttr)&attributes(attributes)

    if (icon && isLeft)
      +icon(name, width, height)

    span.link__text !{text}

    if (icon && !isLeft)
      +icon(name, width, height)

Комбинирование модификаторов

Если так удобней для стилизации адаптива и/или при небольшой вариативности ссылки модификаторы геометрии текста (размер, насыщенность, межстрочный и межбуквенный интервалы) комбинируются. Декоративные модификаторы (цвет, подчёркивание) лучше оставлять отдельно и не комбинировать с другими модификаторами кроме случаев, когда в макете явно выделены типы ссылок

Показать код 
+link({
  modifiers: ["size-s"],
  text: "Link"
})

&--size {
  &-small {
    font-weight: $font-weight-text-bold;
    font-size: $size-text-l;
    line-height: $line-height-text-m;

    @include vp-767 {
      font-size: $size-text-m;
      line-height: $line-height-text-s;
    }
  }
}
