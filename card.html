Card

Карточка

Использование

Можно использовать для любых целей: как фоновую подложку, основу для карточки товара/услуг, для размещения таблиц/изображений и т.д.

Базовый компонент

Показать шаблон 
mixin card(props = {})
-
  const
        tag = props.tag || "div",
        modifiers = props.modifiers ? props.modifiers.map((mod) => "card--" + mod) : [];

#{tag}(class=["card", ...modifiers])&attributes(attributes)
  block
Показать стили 
.card {
  --vertical-space: 0;
  --horizontal-space: 0;
  --card-border-width: initial;
  --card-border-style: initial;
  --card-border-color: initial;
  --card-border-radius: 0;

  padding: var(--vertical-space) var(--horizontal-space);
  border: var(--card-border-width) var(--card-border-style) var(--card-border-color);
  border-radius: var(--card-border-radius);
}
