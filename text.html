Text

Текст

Использование

Любые текстовые блоки, состоящие из одного и более параграфов

Базовый компонент

Показать шаблон 
mixin text(props = {})
-
  const
        content = props.content || [],
        modifiers = props.modifiers ? props.modifiers.map((mod) => "text--" + mod) : [];

div(class=["text", ...modifiers])&attributes(attributes)
  each paragraph in content
    p!= paragraph
Показать стили 
.text {
  --text-space: #{$space-s};

  p {
    margin: 0;
    padding: 0;
  }

  p + p {
    margin-top: var(--text-space);
  }
}

Модификаторы

Если для проекта не предусмотрены токены, используйте жёстко закодированные значения

.text {
  ...
Комбинирование модификаторов

Если так удобней для стилизации адаптива и/или при небольшой вариативности стилей модификаторы геометрии текста (размер, насыщенность, межстрочный и межбуквенный интервалы) комбинируются. Декоративные модификаторы (цвет, трансформация текста) лучше оставлять отдельно и не комбинировать с другими модификаторами кроме случаев, когда в макете явно задана семантика текста
