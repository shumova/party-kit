Title

Заголовок

Использование

Везде, где подходит по стилям (тег не обязательно должен быть h-, дефолтный тег span)

Все параметры типографики по умолчанию наследуются и задаются модификаторами

Базовый компонент

Показать шаблон 
mixin title(props = {})
-
  const
        tag = props.tag || "span",
        text = props.text || "",
        modifiers = props.modifiers ? props.modifiers.map((mod) => "title--" + mod) : [];

#{tag}(class=["title", ...modifiers])&attributes(attributes)!= text
Показать стили 
.title {
  --font-weight: inherit;
  --font-size: inherit;
  --line-height: inherit;
  --letter-spacing: inherit;

  margin: 0;
  padding: 0;
  font-weight: var(--font-weight);
  font-size: var(--font-size);
  line-height: var(--line-height);
  letter-spacing: var(--letter-spacing);
}

Модификаторы

Если для проекта не предусмотрены токены, используй жёстко закодированные значения

.title {
  ...
Комбинирование модификаторов

Если так удобней для стилизации адаптива и/или при небольшой вариативности заголовков модификаторы геометрии текста (размер, насыщенность, межстрочный и межбуквенный интервалы) комбинируются. Декоративные модификаторы (цвет, подчёркивание) лучше оставлять отдельно и не комбинировать с другими модификаторами кроме случаев, когда в макете явно задана семантика заголовков
